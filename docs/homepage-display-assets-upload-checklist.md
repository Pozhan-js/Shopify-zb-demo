## 3. Theme Editor 绑定顺序

建议按这个顺序绑定，最容易看出首页是否恢复完整：

1. `Home Hero`
2. `Product Info Section`
3. `Home Materials`
4. `视频展示区`
5. `Home Endorsements`
6. `Feature Cards Section`

## 4. 绑定完成后的检查

- Hero 轮播应只显示 3 张有效图片
- 商品区切换容量时应能切到对应图片
- 材料区 3 张卡片都有图
- 视频区不再显示上传提示文案
- 评价区左侧头像与右侧大图都能正常切换
- Amazon 商品卡 4 张图片都正常显示
